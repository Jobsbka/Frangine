# Frabgine Engine Build GUI

Простой графический интерфейс для компиляции и тестирования Frabgine Engine.

## Требования

- Python 3.8+
- tkinter (обычно поставляется с Python)
- CMake 3.20+
- G++ 11+ (с поддержкой C++20)
- Vulkan SDK (опционально, для поддержки Vulkan)

## Установка зависимостей

### Debian/Ubuntu:
```bash
sudo apt-get install python3-tk cmake g++ vulkan-sdk
```

### Fedora:
```bash
sudo dnf install python3-tkinter cmake gcc-c++ vulkan-loader-devel
```

### Arch Linux:
```bash
sudo pacman -S tk cmake gcc vulkan-icd-loader
```

## Запуск

```bash
python3 frabgine_builder_gui.py
```

Или сделайте файл исполняемым:
```bash
chmod +x frabgine_builder_gui.py
./frabgine_builder_gui.py
```

## Возможности

1. **Настройка сборки (Configure)**
   - Выбор типа сборки (Debug/Release/RelWithDebInfo)
   - Включение/отключение Vulkan
   - Включение/отключение редактора

2. **Сборка проекта (Build)**
   - Компиляция всех библиотек и исполняемых файлов
   - Отображение прогресса в реальном времени

3. **Очистка (Clean)**
   - Удаление папки build со всеми артефактами сборки

4. **Запуск тестов (Run Tests)**
   - Запуск тестового исполняемого файла
   - Если тесты не найдены, попытка запустить редактор

5. **Мониторинг**
   - Просмотр вывода консоли в реальном времени
   - Индикация статуса выполнения
   - Возможность остановки процесса

## Примечания

- Для сборки требуется наличие всех зависимостей, указанных в CMakeLists.txt
- При отсутствии Vulkan SDK отключите опцию "Включить Vulkan"
- Редактор требует установленный Qt6
- Все логи сохраняются в окне вывода во время выполнения

## Troubleshooting

**Ошибка: "cmake not found"**
- Установите CMake через пакетный менеджер вашей ОС

**Ошибка: "Vulkan not found"**
- Установите Vulkan SDK или отключите поддержку Vulkan в настройках

**Ошибка: "Qt6 not found"**
- Установите Qt6 или отключите сборку редактора

**GUI не запускается**
- Убедитесь, что tkinter установлен: `python3 -c "import tkinter"`
- Проверьте наличие дисплея (для работы GUI)
