# Frabgine Engine

Современный проприетарный игровой движок на C++20, созданный на основе архитектурной философии wz4.

## Архитектура

Движок построен по модульному принципу с четким разделением ответственности:

```
frabgine_engine/
├── core/           # Базовые компоненты (математика, память, сериализация, утилиты)
├── graphics/       # Графическая подсистема (Vulkan рендерер, ресурсы, шейдеры)
├── engine/         # Системы движка (сцена, физика, аудио, анимация)
├── editor/         # Редактор на Qt 6 с узловой системой
├── scripts/        # Скрипты и инструменты сборки
├── build/          # Папка для сборки
└── docs/           # Документация
```

## Ключевые особенности

- **C++20**: Современные возможности языка (умные указатели, корутины, концепты)
- **Vulkan**: Низкоуровневый графический API для максимальной производительности
- **Узловая система**: Процедурная генерация и композиция (материалы, частицы, постобработка)
- **Модульность**: Отдельные библиотеки для каждого компонента
- **Кроссплатформенность**: Поддержка Windows, Linux, macOS

## Компоненты ядра

### Математика (`core/math/`)
- Векторы (Vector2f, Vector3f, Vector4f)
- Матрицы (Matrix44)
- Кватернионы (Quaternion)
- Утилиты (Random, lerp, smoothstep)

### Память (`core/memory/`)
- Умные указатели (UniquePtr, SharedPtr, WeakPtr, RefPtr)
- Пулы памяти для эффективного выделения
- Пользовательские аллокаторы

### Сериализация (`core/serialization/`)
- JSON сериализация (для редактора и отладки)
- Бинарная сериализация (для быстрого запуска игры)
- Endian-safe форматы

### Утилиты (`core/utils/`)
- Timer: Измерение времени
- Logger: Система логирования
- TaskScheduler: Многопоточный планировщик задач
- FileUtils: Работа с файловой системой

## Зависимости

- Vulkan SDK
- PhysX 5.0
- OpenAL Soft
- Qt 6 (Widgets, OpenGLWidgets)
- nlohmann/json
- CMake 3.20+

## Сборка

```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build .
```
